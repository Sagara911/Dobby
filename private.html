🔒 隐私设置

Dobby 默认接了 Cloudflare Web Analytics 看大致访客量(纯统计,不卖数据、不存指纹、不跨站追踪)。如果你不想让自己的访问被算进数字里 — 比如你是站长在测试 — 这页给你一个开关。

排除当前设备

点下面按钮,这个设备 + 这个浏览器以后访问 Dobby 都不会再加载分析脚本(cloudflareinsights.com)。设置存在浏览器 localStorage 里,清缓存会丢。
多设备使用:每个设备访问这个页面一次,点排除按钮,就排除掉这个设备。

Dobby 会发出哪些网络请求

  • HTML / CSS / JS 静态资源: dobby-aih.pages.dev(Cloudflare Pages 自己的 CDN)
  • 分析 beacon: cloudflareinsights.com — 默认开(可在上面关掉)
  • Giscus 留言板 iframe: giscus.app — 只有访问 /messages.html 才加载
  • AI 模型按需下载: huggingface.co(MODNet / RMBG / LaMa)— 只有使用 AI 抠图 / 去水印才下,缓存在 IndexedDB
  • 压缩引擎按需下载: unpkg.com / cdn.jsdelivr.net(mozjpeg / oxipng / AVIF / terser / opentype)— 只有选了这些引擎才下
你处理的文件**完全不上传**,所有图片 / 视频 / 音频 / 字体处理都在浏览器本地完成。
