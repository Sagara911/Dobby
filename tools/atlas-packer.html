🧱 Atlas Packer

MaxRects 紧密打包 · 任意尺寸散图塞进最小图集。跟「精灵图合成」(规整网格 / 帧动画)不同,这个专治尺寸杂乱的 UI / 图标 / 多动画混合素材。

📂 拖入图片(可多选 / ZIP)

PNG / JPG / WebP,任意尺寸
文件名会成为图集里每块的索引名

打包参数

alpha 低于此值的像素当透明 — 调高可裁掉刀光/辉光/描边的半透明杂边(默认 10)。图集大得离谱多半是这个原因
边缘像素向外复制 N 像素,防止缩放采样时邻块渗色(经典 playable 黑边 bug)
两种自由矩形选择策略,填充率会略有差异 — 哪个更小看素材,可两个都试一下对比填充率
按文件名配对:foo + foo_n(也认 _normal / _nrm)。漫反射决定布局,法线图按完全相同的矩形排进第二张图集。导出 Godot 时用 CanvasTexture 绑定 diffuse+normal,2D 法线光照自动对齐

导出

非 PNG 首次使用会联网下载 WASM 编码器(之后缓存)。WebP/AVIF 都带透明通道,Godot 4 / Unity / Cocos / 现代浏览器都原生支持
把 RGB 预乘进 alpha,配合预乘混合可消除半透明边缘的黑/白边(Pixi/Phaser playable 常见)。只有 TexturePacker JSON / libGDX 会写 pma 标记;其它格式会忽略并提示
