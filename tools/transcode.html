🔁 万能转码

一站转码 · 拖文件进来,自动认类型,直接给出目标格式 → 一键转完。所有处理在浏览器本地。

支持的转换

🖼️ 图片:PNG / JPEG / WebP / AVIF 互转
🎬 视频:MP4 / WebM 互转
🔊 音频:任意输入(MP3 / OGG / FLAC / M4A / WAV) → WAV / Opus(WebM)
🎞️ 动图:GIF / APNG(转给 GIF 工具)
📄 其它:base64 / SVG → PNG(转给对应工具)

没支持的格式

❌ HEIC / HEIF(浏览器解码受限)
❌ MP3 编码(没装 LAME,要装 ~100KB 首次加载,看反馈再做)
❌ MKV / FLV(浏览器不直接解封装)
❌ WOFF / WOFF2 字体互转(在 font-subset 里做子集)
拖任意文件进来,Dobby 自动识别并给出转码选项
