🔧 Dobby 还没初始化留言板

  1. 去 GitHub 仓库 Sagara911/Dobby → Settings → Features → 勾选 Discussions
  2. 装一下 Giscus app: https://github.com/apps/giscus → 只授权 Sagara911/Dobby 这一个仓库
  3. 访问 giscus.app/zh-CN → 填仓库名 → 复制出来的 data-repo-id 和 data-category-id 两个值,填到 messages.html 里 GISCUS_REPO_ID / GISCUS_CATEGORY_ID 占位符上,推一下就好
