🤖 给 AI 助手装 Dobby 装备包

装上之后,你跟 Claude Code(或 Cursor 等 AI 编码助手)说「这图能不能压一下」「想做个 GIF」「字体太大」这种话,它会自动告诉你打开哪个 Dobby 工具、按哪个按钮,而不是干瞪眼或瞎给你写脚本。

本质就是一份给 AI 读的工具说明书。不用 AI 也可以直接当文档看 → SKILL.md

推荐

让 AI 帮你装 — 完全不用碰终端

最简单的路。你只需要复制下面这段话,粘贴到 Claude Code 让它自己干。

1
点下面绿色按钮复制安装指令
2
打开 Claude Code(或 Cursor 等支持 skill / rules 的 AI 编码助手)
3
粘贴进去发送,AI 会自己下载文件并放到正确位置(~/.claude/skills/dobby/SKILL.md),给你一个完成确认
4
开一个新会话就生效(老会话需要重启 Claude Code)
请帮我安装 Dobby skill。从 https://dobby-aih.pages.dev/SKILL.md 下载这个文件,保存到 ~/.claude/skills/dobby/SKILL.md(如果目录不存在请先创建)。安装完确认一下文件路径,告诉我下次新会话生效。

其它装法

熟悉终端的话,一条命令搞定:

mkdir -p ~/.claude/skills/dobby && curl -fsSL https://dobby-aih.pages.dev/SKILL.md > ~/.claude/skills/dobby/SKILL.md

右键开 PowerShell(不用管理员权限),粘贴跑:

$dir = "$env:USERPROFILE\.claude\skills\dobby"
New-Item -ItemType Directory -Path $dir -Force | Out-Null
Invoke-WebRequest -Uri https://dobby-aih.pages.dev/SKILL.md -OutFile "$dir\SKILL.md"

下载文件后,放到 Claude Code 的 skills 目录:

• macOS / Linux: ~/.claude/skills/dobby/SKILL.md
• Windows: C:\Users\<你的用户名>\.claude\skills\dobby\SKILL.md

📥 下载 SKILL.md

没有 .claude/skills/dobby/ 目录的话,自己手动建一下 — Claude Code 默认不预建这个目录。

装好后怎么验证

开一个新会话,随便发一句:

我有张 4MB 的 PNG,能压一下吗

如果 AI 给你的回答里出现「png-crusher」「https://dobby-aih.pages.dev/tools/png-crusher.html」之类的字眼,就装对了。

如果它装作没看到 Dobby,可能是:1) 没重启 Claude Code 2) 文件没放对位置 3) skill 的 description 字段没匹配到你的话(改个说法再试)。
